## RustLexTest
A simple lexer written in Rust for testing purposes.

#### Supports
- Identifier keywords (defined **[here](https://github.com/x-Eagle-x/RustLexTest/blob/main/lex.rs#L48)**)
- Escape sequences (`\r`, `\n`, `\t`)
